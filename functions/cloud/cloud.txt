Basically a way for you to store a variable to be loaded automatically by JIP clients.

Cloud Variable (select 0) - Identifier for the data. //String
Cloud Data (select 1) 	  - The data of the cloud item. //Data
Cloud Key (select 2) 	  - Anything you want, it's a way to identify multiples of the same kind if you choose. //String

cloud_Add -
Adds a part into the cloud. Overwrites if exists.

cloud_Retreive -
Gets the index of a cloud piece indexed by variable name.

cloud_Delete -
Delete a cloud piece by either variable name or key.

cloud_Publish -
Publishes the cloud to the given unit, provided they are part of the network.

cloud_SendToClient -
Sends requested variable to given unit, provided they are part of the network.
